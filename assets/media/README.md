# Media Folder Guide

- `assets/media/videos/`: all mp4 video assets used by pages.
- `assets/media/images/`: all local image assets used by cart and catalog.

## Naming

- Keep original product/chapter names when possible.
- Prefer `kebab-case` for new files.

## Path usage

From root HTML pages, use:

- Video: `assets/media/videos/<file>.mp4`
- Image: `assets/media/images/<file>.<ext>`
